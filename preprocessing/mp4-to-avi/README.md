# MP4 to AVI module

## Cardiac Tissue Image Analysis – Preprocessing Module
This repository includes an automated tool to convert MP4 format microscopy videos to uncompressed AVI format using the VirtualDub2 and AutoHotkey tools.
This process is necessary to make the videos compatible with the subsequent image processing steps.

---

## Purpose
Videos generated by microscopy systems are often stored in the MP4 format.
This format can lead to problems such as:
- Compression artifacts
- Frame inconsistencies
- Lack of compatibility with FIJI and MATLAB
- Inaccurate pixel-level quantitative data
To eliminate such problems and ensure the integrity and reproducibility of the video frames, the video is converted to the **Uncompressed AVI format**.

---

## Requirements
- Windows OS
- AutoHotkey (English version)
  https://www.autohotkey.com/
- VirtualDub2

---

## Installation
1. Install AutoHotkey (English version)
2. Install VirtualDub2
3. Copy the file **VirtualDub2_batch.ahk** to your desired working directory

---

### Usage
1. Start **VirtualDub2**, and make sure the window is maximized.
2. Press keys manually to check that the following work correctly:
   - `Ctrl + O`: Opens the "Open video" dialog box.
   - `Alt + V`: Opens the "Video" menu.
   - Selects "Compression…."
   - Selects "Uncompressed."
   - Saves the video with `F7`.
3. Double-click on the `VirtualDub2_batch.ahk` file.
4. Select the directory containing the MP4 files.
5. Wait until the processing is done.
*Do not touch VirtualDub2 until the processing is done.

### Output
A new directory containing the following files:
- Uncompressed AVI files
- Original MP4 files

### Limitations
- Only tested on Windows.
- It is based on UI automation, which requires the VirtualDub2 window to be on top of all windows.
- It may not work on monitors with low resolutions or non-standard layouts.
- It is not optimized for performance-critical batch processing.

### Role in the Full Pipeline
This tool is part of the overall **Cardiac Tissue Image Analysis Pipeline**, which is defined as follows:
MP4 -> AVI (Uncompressed) -> FIJI Processing -> MATLAB Quantification

### Reproducibility Note
This tool was developed to standardize video processing across all the experiments to ensure reproducibility of results in the field of cardiac tissue image analysis.
